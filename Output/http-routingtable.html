HTTP Routing Table

/caches | /cms | /shops | /states
 
/caches
GET /caches null
GET /caches/{cacheName} null
DELETE /caches/{cacheName} null
 
/cms
GET /cms (beta) Returns the list of CMS sub resources
GET /cms/includes (beta) Returns a list of all includes
GET /cms/includes/{includesKey} (beta) Returns a specific include
GET /cms/pagelets (beta) Returns the list of all published pagelets of the domain
GET /cms/pagelets/{pageletsKey} (beta) Returns a specific pagelet of the domain
GET /cms/pagelets/{pageletsKey}/{slotsKey} (beta) Returns a specific slot of a pagelet
GET /cms/pages (beta) Returns a list of all pages
GET /cms/pages/{pagesKey} (beta) Returns a specific page
GET /cms/pagetree (beta) Returns all pages with their tree hierarchy
GET /cms/pagetree/{pagesKey} (beta) Returns a specific page hierarchy
GET /cms/viewcontexts (beta) Returns all view contexts of the domain
GET /cms/viewcontexts/{viewContextID} (beta) Returns a specific view context of the domain
GET /cms/viewcontexts/{viewContextID}/entrypoint (beta) Resolves a page for a view context
 
/shops
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests List the return requests
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/returnables List the returnable order positions to prepare the creation of a return request
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId} Get a return request
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/contact-persons List the return request contact persons
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/custom-attributes List the return request custom attributes
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/pickup-addresses List the return request pickup addresses
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/positions List the return request positions
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/positions/{returnRequestPositionId} Get a return request position
GET /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/positions/{returnRequestPositionId}/items List the return request position items
GET /shops/{shopName}/return-reasons List the configured return reasons for a given shop
POST /shops/{shopName}/orders/{shopOrderNumber}/return-requests Create a return request
POST /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/approvals Create an approval
 
/states
GET /states Get orders states by search criteria
