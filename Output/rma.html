Intershop Order Management RMA API¶

GET /shops/{shopName}/return-reasons¶

List the configured return reasons for a given shop

Returns a list of return reasons configured for a given shop.

Parameters

  • shopName (string) – The shop name

Query Parameters

  • type (array) – Filter the return reasons by name of 1..n return types

Status Codes

  • 200 OK – OK

  • 400 Bad Request – Bad Request

  • 401 Unauthorized – Authentication information is missing or invalid

  • 403 Forbidden – Forbidden

  • 404 Not Found – Not Found

POST /shops/{shopName}/orders/{shopOrderNumber}/return-requests/{returnRequestId}/approvals¶

Create an approval

Creates a new approval

Parameters

  • returnRequestId (integer) – The return request id of the related order

  • shopOrderNumber (string) – The order number of the shop

  • shopName (string) – The shop name

Status Codes

  • 201 Created – Created

  • 400 Bad Request – Bad Request

  • 401 Unauthorized – Authentication information is missing or invalid

  • 403 Forbidden – Forbidden

  • 415 Unsupported Media Type – Unsupported Media Type

  • 500 Internal Server Error – An unexpected error occured

Response Headers

  • Location – The path to the newly created resource
