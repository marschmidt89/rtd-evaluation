Intershop Order Management Order State API¶

GET /states¶

Get orders states by search criteria

Obtain the order states of the orders which are matching the given search criterias

Query Parameters

  • shopId (integer) – The id of the IOM shop instance (Required)

  • orderNumber (array) – Order number as used by the shop

  • customerNumber (array) – Customer number as used by the shop

  • productNumber (array) – Product number as used by the shop

  • orderState (array) – Order state as used by the shop if configured, otherweise the IOM internal names of order states

  • orderCreationDate[gte] (string) – The earliest ‘shop order creation date’ of an order

  • orderCreationDate[lte] (string) – The latest ‘shop order creation date’ of an order

  • orderBy (string) –

  • sortDirection (string) –

  • offset (integer) – The starting row (index) of the first element to return

  • limit (integer) – The number of items to return

Status Codes

  • 200 OK – OK

  • 400 Bad Request – Bad Request

  • 401 Unauthorized – Authentication information is missing or invalid

  • 403 Forbidden – Forbidden

  • 406 Not Acceptable – Not Acceptable

  • 500 Internal Server Error – An unexpected error occured
